3M Telesteth

Redesign the interface of 3M Littmann TeleSteth system.



 Type: Group Project
Time: October 2015
~ Brainstorming, Ideation, Prototyping ~

Overview

Project Goal

The goal of the project was to redesign the existing interface of 3M Littmann TeleSteth system. The application connects remote health centers and doctors over internet to carry out quick diagnosis. 3M TeleSteth Stethoscope is a bluetooth enabled smart stethoscope, measures multiple vital signs and records into the application. With growing number of users across the country, 3M wanted a more user-friendly, structured and intuitive interface to engage users with any level of expertise.


Solution

Our solution was focused on providing a simple platform with better user experience and smoother information flow. We concentrated on incorporating minimum texts images and fields to reduce cognitive load. Considering the health center helpers are technically novice users, an intuitive and easy to use interface is necessary in order to carry out special diagnostic tasks.


Role

My role was to create user flow and design user interface. I sketched out several options for the communication screen and presented to the team to stream out the best one.


Process

Brainstorming and Prototype

Given the short time-frame, brainstorming and bodystorming brought out ideas to make the interface more user-centric and intuitive. We focused on revamping the few main features of the application with smooth transition and better experience.
Login and connecting with the right doctor/helper.
On-boarding for new users.
Include voice, video and text chat feature.
Generate and download diagnosis report for record.
